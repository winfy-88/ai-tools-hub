# 电影资源网站参考

## 英文资源站

### YTS / YTS MX
- **主页**: https://yts.mx
- **搜索**: https://yts.mx/browse-movies/{title}-{year}
- **特点**: 界面最干净，质量最稳定，720p/1080p/4K 任选，magnet 链接直接
- **筛选**: 可按分辨率 / 类型 / 评分 / 年份筛选
- **注意**: 部分国家地区可能需要代理访问

### 1337x
- **主页**: https://1337x.to
- **搜索**: https://1337x.to/search/{title}/{year}/1/
- **特点**: 资源数量最多，更新最快，热门新片第一时间有
- **注意**: 广告多，注意辨别假下载按钮

### The Pirate Bay
- **主页**: https://thepiratebay.org
- **搜索**: https://thepiratebay.org/search.php?q={title}+{year}
- **特点**: 最大的 P2P 追踪器，种子数最多
- **镜像**: 经常换域名，搜索 "TPB 镜像" 获取最新地址

### TorrentGalaxy
- **主页**: https://torrentgalaxy.to
- **搜索**: https://torrentgalaxy.to/torrents.php?search={title}
- **特点**: 活跃社区，高清资源多，种子健康度高

### RARBG (镜像)
- **特点**: 高质量压制，HEVC 资源丰富
- **状态**: 原站已关闭，搜索可用 RARBG 镜像站

### Zooqle
- **主页**: https://zooqle.com
- **特点**: 元数据质量高，支持按 IMDB 搜索

### BT4G
- **主页**: https://bt4g.org
- **特点**: 聚合 DHT 搜索引擎，免注册，支持磁力链接搜索

## 中文资源站

### 电影天堂 (dytt8)
- **主页**: http://www.dytt8.com
- **特点**: 国内用户最常用的电影站，高清资源多，中文字幕
- **搜索**: http://www.dytt8.com/search.asp?searchword={关键词}

### 高清之家
- **主页**: https://www.gaoxiaozhijia.com
- **特点**: 专注高清电影（4K/蓝光），字幕齐全

### 4K 电影下载
- **主页**: https://www.4kdy.com
- **特点**: 专注 4K HDR 资源，HEVC 编码

### 迅雷电影天堂
- **主页**: http://www.xl720.com
- **特点**: 迅雷链接和磁力链接都有

### 飘花资源网
- **主页**: https://www.piaohua.com
- **特点**: 更新快，资源全，支持在线播放

## 搜索语法技巧

```
# 限定分辨率
"{title}" 1080p WEB-DL
"{title}" 4K HDR
"{title}" BluRay 2160p

# 限定编解码
"{title}" HEVC x265  # 体积小画质好
"{title}" H264 x264   # 兼容性好

# 限定语言
"{title}" 中文字幕 1080p
"{title}" 双语字幕

# 限定年份（排除重名）
"{title}" 2024
"{title}" 2023 1080p
```
