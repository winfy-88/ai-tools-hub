## 🚀 快速开始

### 在线使用
直接访问 **[https://winfy-88.github.io/ai-tools-hub/](https://winfy-88.github.io/ai-tools-hub/)** 即可使用，无需安装任何东西。

### 本地部署
```bash
# 克隆仓库
git clone https://github.com/winfy-88/ai-tools-hub.git

# 直接用浏览器打开 index.html 即可
# 无需构建、无需依赖，纯静态页面
```

---

## 🛠️ 添加新工具

直接在 `index.html` 中添加新的 `.tool-card` 区块，参考现有格式：

```html
<div class="tool-card" data-cat="chat" data-name="tool-name">
  <div class="card-top">
    <div class="tool-icon" style="background:linear-gradient(135deg,#6c5ce7,#a29bfe)">🤖</div>
    <div class="card-info">
      <div class="tool-name">Tool Name</div>
      <div class="tool-tagline">一句话介绍这个工具</div>
    </div>
  </div>
  <div class="card-tags">
    <span class="tag tag-freemium">Freemium</span>
  </div>
  <div class="card-footer">
    <span class="pricing freemium">免费版可用</span>
    <a href="https://example.com" target="_blank" class="visit-btn">立即使用 →</a>
  </div>
</div>
```
